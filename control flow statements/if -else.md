# if-else

**syntax**

>if(condition)
>{
>
>if statement
>
>}
>else{
>
>else statement
>
>}
>
**example**
>const maximum_value = 100;
>
>let counter = 100;
>
>if (counter < maximum_value) {
>
>  counter++;
>
>} else {
>
>counter = 1;
>
>}
>
>console.log(counter);


**Ternary operator ?:**

> condition?true:false
